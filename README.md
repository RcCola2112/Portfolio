# BSIT Portfolio - Rc Denver

This repository is a frontend-only portfolio scaffold for a BSIT student specializing in full-stack web and mobile development. Built with Next.js + Tailwind CSS.

Quick start

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
npm run start
```

Notes and environment

- Recommended Node.js: 18.x or newer.
- The project uses Next.js (pages router) and Tailwind CSS.
- `public/` and `assets/` can contain images — currently placeholders are used.

Project structure

- `components/` - reusable React components
- `pages/` - Next.js pages
- `styles/` - global styles and Tailwind
- `public/` or `assets/` - static images (use placeholder links currently)

Notes

- This is frontend-only. Contact form is UI-only.
- Update content, replace placeholder images, and link live demos/code as needed.
